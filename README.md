# GDCxTripoAI Game Jam – Team PingTitans

## Overview
The world is threatened by a dangerous new virus called **Cobid-26**.  
You play as a scientist whose mission is to stop the spread of the virus.  
Eliminate virus particles by stepping on them before they accumulate and take over the lab.

## Gameplay
- Move the player using **WASD** or **Arrow Keys**.
- Approach the **red virus particles** and step on them to eliminate them.
- The game becomes more challenging as the **virus spawn rate increases** over time.
- Your goal is to **achieve the highest score possible within 30 seconds**.

## Features
- Score system showing how many virus particles were eliminated.
- **3D assets generated using Tripo AI**.
- Start and end menu screens.
- Background music for a more immersive experience.

## Controls
| Action | Key |
|------|------|
| Move | WASD / Arrow Keys |
| Eliminate Virus | Step on virus |

## Technology Used
- Unity Game Engine  
- Tripo AI (for 3D assets)

## Team
Team **PingTitans**

## Objective
Survive the outbreak and eliminate as many virus particles as possible before time runs out.
